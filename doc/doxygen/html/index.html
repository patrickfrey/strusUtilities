strusUtilities program loader

The project strusUtilities provides some command line programs to create, delete and access a strus information retrieval collection storage. It also provides a library to load data into strus define in configuration programs in a specific syntax. The interface of this program loader library is introduced here.

Programs for analyzer configuration:

  • Load document analyzer program
  • Load query analyzer program
  • Define a phrase type for the query analyzer by its subparts

Programs for query evaluation configuration:

  • Load a query evaluation program

Programs for parsing a query from source (query language implementation):

  • Load a query from source

Scan a source file containing a list multiple programs:

  • Scan the next program source

Loading storage contents from file:

  • Load global statistics from file
  • Load meta data assignments from file
  • Load document attribute assignments from file
  • Load user right assignments from file
